## DESIGN GUIDELINES

### COLOR PALETTE (Warm Terracotta Theme)

**Primary:** #05087c (Blue - Identity, Strength)  
**Secondary:** #E7E8D1 (Sand - Clean, professional)  
**Accent:** #A7BEAE (Sage - Growth, community)  
**Dark:** #2F3C7E (Navy - Authority, trust)  
**Black:** #212121 (Text)  
**White:** #FFFFFF (Backgrounds)

### TYPOGRAPHY

**Headings:** Bold, modern sans-serif (e.g., Poppins, Montserrat, Inter)  
**Body:** Clean, readable (e.g., Open Sans, Roboto, System Font)

**Font Sizes (Desktop):**
- H1: 48-56px
- H2: 36-42px
- H3: 28-32px
- Body: 16-18px
- Small: 14px

**Font Sizes (Mobile):**
- H1: 32-36px
- H2: 24-28px
- H3: 20-24px
- Body: 16px
- Small: 14px

### IMAGERY

**Style:** High-energy, authentic, community-focused  
**Examples:**
- Artists creating (painting, performing, designing)
- Workshop/conference moments
- Egbeda community vibes
- Cultural elements (traditional + modern fusion)

**Avoid:** Generic stock photos, overly polished/corporate imagery

### BUTTONS & CTAs

**Primary Button (GET TICKETS, etc.):**
- Background: Blue (#05087c)
- Text: White
- Hover: Darker terracotta or scale animation
- Size: Large, prominent, easy to tap on mobile

**Secondary Button (Learn More, etc.):**
- Border: Terracotta
- Text: Terracotta
- Background: Transparent or white
- Hover: Filled terracotta

### SPACING & LAYOUT

- Generous white space (avoid clutter)
- Mobile-first: All content readable on small screens
- Consistent padding: 20-40px sections
- Centered content with max-width ~1200px

## TECHNICAL REQUIREMENTS

### MUST-HAVES

1. **Mobile-Responsive**
   - Mobile-first design (80%+ traffic will be mobile)
   - Touch-friendly buttons (min 44x44px)
   - Readable text without zooming

2. **Ticketing Integration**
   - Direct links to ticketing platform
   - OR embedded iframe (if platform supports)
   - OR API integration for seamless checkout

3. **Countdown Timer**
   - Dynamic JavaScript countdown to March 20, 2026
   - Shows: Days, Hours, Minutes
   - Updates in real-time

4. **Social Media Integration**
   - Share buttons on key pages (Homepage, Artists)
   - Social media feed embed (Instagram, optional)
   - Meta tags for social sharing (Open Graph)

5. **Fast Loading**
   - Optimized images (WebP format, lazy loading)
   - Minimal JavaScript (use native features when possible)
   - Page load under 3 seconds on mobile

6. **Analytics**
   - Google Analytics or equivalent
   - Track: Page views, ticket button clicks, form submissions

7. **SEO Basics**
   - Page titles, meta descriptions
   - Alt text for images
   - Structured data (Event schema)

### NICE-TO-HAVES

1. **Email Capture**
   - Newsletter signup (MailChimp, etc.)
   - Pop-up or inline form

2. **Instagram Feed**
   - Live Instagram posts from @conformlagos

3. **WhatsApp Integration**
   - Click-to-WhatsApp button for quick contact

4. **Progressive Web App (PWA)**
   - Offline access, add to home screen
